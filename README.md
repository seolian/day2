1. 프로젝트 개요
본 프로젝트는 파이썬 기반의 마우스 및 키보드 자동화 도구입니다. 반복적인 클릭이나 키 입력 작업을 자동화하여 업무 생산성을 높이기 위해 개발되었습니다. GUI 환경을 제공하여 누구나 직관적으로 설정하고 사용할 수 있습니다.

2. 시스템 요구 사항
필수 라이브러리:
pyautogui: 마우스 제어 자동화
keyboard: 키보드 단축키 감지 및 제어

설치 명령어:
pip install pyautogui keyboard

3. 프로그램 실행 가이드
파이썬 설치 확인: python --version

소스 코드 저장: auto_tool.py 파일로 저장

터미널 실행: 파일이 위치한 폴더에서 python auto_tool.py 입력

4. 상세 제어 방법
4.1 GUI 설정 항목
마우스 위치: '현재위치' 또는 '특정위치(X, Y)' 설정 가능

동작 선택: 좌/우/더블클릭, 텍스트 입력, 단축키 조합 지원

실행 간격: 자동화 동작 간의 대기 시간(초 단위) 설정

4.2 전역 단축키 (Global Hotkeys)
프로그램이 백그라운드에 있어도 즉시 작동합니다.

F8: 자동화 시작

F9: 자동화 일시 정지

Esc: 프로그램 즉시 종료

5. 기술적 특징 및 아키텍처
멀티스레딩(Multi-threading): 작업 루프를 별도의 스레드로 분리하여, 자동화 작업 중에도 GUI 버튼(정지, 종료 등)이 반응하도록 설계되었습니다.

방어적 프로그래밍: try-except 구문을 적용하여 비정상적인 입력값(문자 등)이 들어와도 프로그램이 중단되지 않고 안정적으로 작동합니다.

6. 안전 수칙 및 주의사항
관리자 권한: 일부 프로그램(게임, 보안 소프트웨어 등)에서 사용 시 반드시 관리자 권한으로 터미널을 실행하십시오.

긴급 정지: 오작동 시 당황하지 말고 즉시 Esc를 누르거나, 마우스를 화면 좌측 상단(0,0)으로 이동(Fail-safe)시키십시오.

간격 설정: 0.1초 미만의 너무 짧은 간격은 시스템 렉을 유발할 수 있으므로 권장하지 않습니다.
